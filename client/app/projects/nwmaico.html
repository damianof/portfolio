NW Maico & CZ

NW Maico & CZ was an online store for a motorcycle parts company. The application was build using KnockoutJS, a popular javascript MV* library that provides data-binding via "observables". I also used SammyJS to give the application single-page hash-navigation. Purchasing was handled with PayPal.

The application was developed as part of a planned deal that fell through. Unfortunately, this meant that the application was never deployed to production, though it did go through beta testing.

After the initial release, the web application was expanded to give reporting capabilities to our clients.
